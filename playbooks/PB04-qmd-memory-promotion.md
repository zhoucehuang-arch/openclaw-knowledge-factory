# PB04 QMD 记忆升级剧本

## Goal
将高频情报转为可执行知识，减少重复学习。

## Trigger
出现“报告很多但行动少”或“同主题反复搜索”。

## Steps
1. 从最近 48h 报告抽取候选事实。
2. 执行去重并按主题聚类。
3. 每个主题提炼 1-2 条 principle。
4. 每条 principle 绑定 1 个可执行 action。
5. 一周后回填 outcome，更新优先级。

## Metrics
- 主指标：principle->action 转化率 > 60%
- 护栏：无证据 principle 数量 = 0

## Rollback
若误提炼过多，降级为 facts 并重新验证。
